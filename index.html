Bootstrap 4 - Shopping Cart

Easy to add to a new or existing project. Requires Bootstrap 4 Css & Js, Jquery, and Fontawesome. Just add the resources */style.css and */script.js to get going.

Customize the layout to your own preference. All you need is to provide the Add to Cart buttton with the data-product_id, data-pimage data-name, and data-price attributes.

The cart form data is generated with the function displayCart() in the script.js code. You can modify it to your own preference.
